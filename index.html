Projects

Stella

2017

I helped develop Stella as part of Launchpad last semester. Stella is a Chrome extension that adds a voice assistant to your browsing session. With it, users can do things like look up music, open and close tabs, and otherwise interact with their browser; all from across the room. This project uses a Naive Bayes algorithm to figure out how to act on user queries, and we plan on adding a recurrent neural network to improve on the speech-to-text component in the near future.

Quadris

2016

This is a self-playing Tetris agent which learned to play through a Genetic Algorithm. I built both the learning algorithm and the Tetris game as a final project with my friend in our AP computer science class. I choose this project because I wanted to learn more about machine learning and I thought it would be good to get real-world experience. After about 15 epochs of training, our algorithm outperformed humans on average, though it had a significant standard deviation in its performance.
